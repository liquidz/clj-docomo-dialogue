# clj-docomo-dialogue

## Usage

```clj
(require '[clj-docomo-dialogue.core :refer :all])

(def api-key "YOUR DOCOMO API KEY")

(talk api-key "こんにちは")
(talk api-key "こんにちは" {:t 30})
(talk api-key "こんにちは" {:context "CONTEXT"})
```
